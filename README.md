# TIL (Today I Learned)

매일 배운 내용을 주제별로 정리하는 저장소입니다.

## 구조

- `AI/`: LLM, LangChain, LangGraph 등 인공지능 관련
    - [langgraph-router-vs-agent.md](AI/langgraph-router-vs-agent.md): LangGraph를 이용한 챗봇 구현 시 Router 방식에서 Agent 방식으로의 진화 과정 정리
- `Algorithm/`: 알고리즘 문제 풀이 및 이론
- `DevOps/`: Docker, Kubernetes, CI/CD 등
- `Language/`: Java, Python, JavaScript 등 언어 관련
    - [python-closure.md](Language/Python/python-closure.md): 파이썬 클로저(Closure)의 개념, 사용 이유 및 동작 원리 정리
- `Network/`: 네트워크 이론 및 실습
- `OS/`: 운영체제 관련
- `Web/`: 프론트엔드, 백엔드 웹 기술

## 작성 규칙

1. 해당 주제 폴더 내에 마크다운 파일(`.md`)로 작성합니다.
2. 파일명은 내용을 잘 나타내는 이름으로 정합니다. (예: `docker-compose-basics.md`)
3. `README.md`의 각 주제 섹션에 해당 파일에 대한 하이퍼링크와 한 줄 정도의 요약을 추가합니다.
4. 꾸준히 작성하는 것을 목표로 합니다!
